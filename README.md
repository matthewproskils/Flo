# Basic Syntax:

### **Functions**
##### Defining
```
fun <name> (args*) {
 <code>
} 
```
##### Calling
```
<name>(args*)
```
### *Variables*
#### Defining
```
name = "Foo"
```
#### Calling
```
name
```
#### Lines Split By `,pls,`
```
,pls,
```
