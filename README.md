# FateLang
Made by CodeLongAndProsper, MatthewProSkills, and PowerCoder

### **`Functions`**
##### Defining
```
fun <name> (args*) {
 <code>
} 
```
##### Calling
```
<name>(args*)
```
### **`Variables`**
#### Defining
```
name = "Foo"
```
#### Calling
```
name
```
#### Lines Split By `,pls,`
```
,pls,
```
### **`Space Manipulators`**
```
\n(<amm>)
```
Specifies the new line ammount.
```
\s(<amm>)
```
Specifies the space ammount.
```
\t(<amm>)
```
Specifies the tab ammount.
